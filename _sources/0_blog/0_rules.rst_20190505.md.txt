# Blog Rules

## File Naming Rules
[Index]\_[Name]\_[LastUpdatedDate(YYYYMMDD)]

- Index: 디렉토리 내 에서 문서가 생성된 순서대로 증가함. 한 번 정해지면 변하지 않음.
- Name: 문서를 나타내는 이름
- LastUpdatedDate: 마지막 수정 날짜. Index는 값이 정해지면 변하지 않는 것에 반해 이 값은 문서가 수정 될 때 마다 변한다.

## Image Insert Link Rule
[DocumentFullPath]\_[Index]\_[Name]\_[LastUpdatedDate(YYYYMMDD)]

- DocumentFullPath: 이미지가 삽입되는 링크의 Full 경로
- Index : 문서의 Index와 같은 개념. Increase Only 값이다.
- Name: 이미지를 나타내는 이름
- LastUpdatedDate: 마지막 수정 날짜. Index는 값이 정해지면 변하지 않는 것에 반해 이 값은 문서가 수정 될 때 마다 변한다.
